Artnet Lantern

Designed and developed software and hardware for an Artnet Lantern. The lantern was designed to receive Artnet messages over WiFi to change the colour of addressable LEDs.

This project was built using ESP8266, and WS2812 LEDs, and included software development, and PCB design and assembly.
