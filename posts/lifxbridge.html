LifxBridge is a software tool that connects LIFX lights to a professional lighting console over the Artnet protocol.

This software was developed using the Qt framework to allow cross-platform builds and makes use of the publicly documented LIFX LAN API to communicate with the lights.

LifxBridge automatically polls and discovers LIFX lights on the network, allowing easy configuration and connection to a lighting console.
