Wireless Artnet Lighthouse

Samuel Black,Wed Jun 01 2022•hardware & software
Back

Designed and developed, both software and hardware for a fully wireless, battery-powered lighthouse to be used in the Counterpilot theatre production of Adrift.

The lighthouse was built using ESP8266 running ESP-IDF to control 32 LED's each controlled individually with the use of shift registers. This required software-based PWM control to be able to dim each individual led.

The lighthouse was designed to listen for Artnet messages over wifi and allowed configuration using my custom configurator software.

The hardware was designed using KiCad and custom rigid and flex PCBs were produced for the final product.
